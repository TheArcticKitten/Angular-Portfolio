Recent Projects: r/SSBM Reddit Bot Python, JSON A bot written in Python which scrapes the r/SSBM subreddit on Reddit.com to look for specific queries that the community might use. Super Smash Brothers Melee is a complicated game with many techniques and play-styles, therefore it has a steep learning curve. The bot helps to flatten this learning curve by providing links to tutorials, a GIF file of the technique in action, and a wiki page for the technique in question. This information is presented as a comment in the thread which the bot detected as a query.
